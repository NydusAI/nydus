Helped a team evaluate EKS vs ECS for their microservice migration.
They have Kubernetes experience from a previous employer, so EKS makes
more sense despite the higher operational overhead. Recommended starting
with a managed node group to reduce initial complexity.

Also answered a question about Lambda cold starts — the user was seeing
800ms+ cold starts on a Java Lambda. Suggested provisioned concurrency
or switching to a lighter runtime (Node.js or Python) for their API
gateway integration.
